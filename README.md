# User Management Project

## How to run it

### `npm install`

### `Configurate ./config/db.config.js`

Add username, password and databse of your local mysql server

### `npm run migrate`

creates a migration from the model to the database

### `npm run start`

Runs the server in the development mode.<br />
It runs in [http://localhost:8080]

Postman collection is in postman_collection.json

### `npm run lint`

To use linter in the project.
